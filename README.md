# Landing Page - Site Web Premium

Une landing page moderne et conversion-orientée avec un design premium, construite avec Next.js et Tailwind CSS.

## Caractéristiques

- **Design moderne** - Interface utilisateur avec des effets visuels captivants
- **100% Responsive** - Optimisé pour tous les appareils (mobile, tablette, desktop)
- **Performance optimale** - Construit avec Next.js pour des temps de chargement ultra-rapides
- **Conversion-oriented** - Éléments de design pensés pour maximiser les conversions
- *Animations fluides** - Transitions et effets hover pour une expérience utilisateur immersive
- **Accessible** - Respect des standards d'accessibilité web

## Technologies utilisées

- **[Next.js](https://nextjs.org/)** - Framework React pour la production
- **[Tailwind CSS](https://tailwindcss.com/)** - Framework CSS utility-first
- **[React Icons](https://react-icons.github.io/react-icons/)** - Bibliothèque d'icônes
